# Open Data AI Analytics

## Мета проєкту

Основною метою проєкту є проведення аналітичного дослідження відкритих державних даних про кількість закладів загальної середньої освіти в Україні з подальшим виявленням часових змін, територіальних відмінностей та прогнозування можливих тенденцій розвитку.

---

## Джерело даних

Портал відкритих даних України:  
https://data.gov.ua/

Обраний датасет:  
**Мережа та діяльність закладів освіти**  
(Державна служба статистики України)

Посилання на набір даних:  
https://data.gov.ua/dataset/3d4a09aa-f296-42c6-9096-7a694fdb772a

Формат даних: XLSX / JSON

---

## Дослідницькі гіпотези

1. Кількість закладів загальної середньої освіти змінюється з роками.
2. Регіони України суттєво відрізняються за кількістю шкіл.
3. Після 2022 року відбулися помітні зміни в освітній інфраструктурі.

---

## Структура проєкту

Проєкт складається з чотирьох модулів:

- **data_load** — завантаження та підготовка даних  
- **data_quality_analysis** — перевірка якості даних (пропуски, дублікати, форматування)  
- **data_research** — аналітичне дослідження  
- **visualization** — побудова графіків та візуалізація результатів  

Кожен модуль розробляється в окремій гілці та після завершення об'єднується з гілкою `main`.
